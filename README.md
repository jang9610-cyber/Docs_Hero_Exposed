## 🎮 게임 한 줄 소개

```
플레이어 = 마왕군 기업(마왕물산) 보안 데스크 신입 주임
목표     = 출근하는 직원 중 위장 잠입한 용사를 자유 채팅 심문으로 가려내기
핵심 루프 = 심문 → PASS/EJECT 판정 → 경제 보상 → 일일 정산 → 멀티 엔딩
```
