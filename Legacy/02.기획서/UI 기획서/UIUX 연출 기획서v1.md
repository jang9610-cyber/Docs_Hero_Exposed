**1. 글로벌 연출 규칙 (Global Presentation Rules)**

본 항목은 게임 내 모든 씬(Scene)에 공통으로 적용되는 시스템 연출 기준을 정의함.

**1.1. 화면 전환 트랜지션 (Fade In / Out)**

- **기본 전환:** 화면 전체 블랙 아웃(Black-out) 형태의 페이드 효과 사용.
- **소요 시간:** Fade-out (암전) 0.5초 / Fade-in (밝아짐) 0.5초.

**1.2. BGM 및 SFX 기본 볼륨/페이드 설정**

- **오디오 전환 (Crossfade):** BGM 변경 시, 기존 트랙 1.0초 Fade-out / 신규 트랙 1.0초 Fade-in 동시 적용.
- **반복 SFX 처리:** 사원증 터치, 도장(PASS) 등 짧은 주기로 반복되는 SFX는 기계적 반복음 방지를 위해 재생 시마다 Pitch(음높이) 값을 0.9 ~ 1.1 사이에서 랜덤하게 적용.

**2. UI 및 캐릭터 상호작용 연출 (Interaction & Animation)**

**2.1. 텍스트 출력 방식 (Typewriter & Skip)**

- **출력 방식:** 한 글자씩 순차적으로 출력되는 타이핑(Typewriter) 연출 적용.
- **기본 속도:** 글자당 0.05초 간격으로 출력.
- **스킵 판정:** 타이핑 중 클릭 시 전체 텍스트 즉시 동시 출력. 전체 출력 후 클릭 시 다음 대사 노출.

**2.2. 버튼 호버(Hover) 및 클릭 연출**

- **일반 UI 버튼 (설정, 홈 버튼 등):**
  - Hover In: 원본 대비 1.1배 확대 및 불투명도 증가 (소요 시간: 0.3초, Ease-out).
  - Hover Out: 0.3초에 걸쳐 원본 상태 복구.
  - Click: 순간적으로 0.95배 축소 후 즉시 복구 (물리적 눌림 타격감 부여).
- **투명 상호작용 영역 (사원증, 도장, 매뉴얼 등 화면 내 오브젝트):**
  - Hover In: 마우스 오버 시 해당 기물에 맞는 반투명한 배경 피드백(Red/Cyan/Yellow 10%) 및 얇은 테두리(Border)가 생성되어 조작 가능함을 암시 (소요 시간: 0.15초 ~ 0.3초).
  - Hover Out: 즉시 투명 상태로 복구.

**2.3. 모달/팝업 창 공통 연출 (플로팅 UI)**

환경설정, 사원증/허가증, 매뉴얼 등 화면 위에 띄워지는 모든 플로팅 UI의 공통 규칙.

- **등장 (Open):** \* 배경 Dim: 투명도 0% ➡️ 60% 페이드 인 (0.2초)
  - 팝업 본체: 크기 0.95배 ➡️ 1.0배로 확대되며 페이드 인 (0.2초, Ease-Out-Quad)
- **퇴장 (Close):**
  - 배경 Dim: 투명도 60% ➡️ 0% (0.15초)
  - 팝업 본체: 크기 1.0배 ➡️ 0.95배로 축소되며 페이드 아웃 (0.15초, Ease-In)

**2.4. 캐릭터 상태별(감정) 애니메이션 및 텍스트 연출**

심문 과정에서 캐릭터의 감정 상태(State)에 따라 모션과 텍스트 출력 방식이 다르게 연출됨.

- **State 01. Normal (기본/대기):**
  - 모션: 호흡 (2~3초 주기로 Y축 기준 미세하게 둥실거림)
  - 텍스트: 0.05초 간격으로 일정하고 차분하게 출력. 기본 폰트.
- **State 02. Flustered (당황/동공지진):**
  - 모션: 미세 진동 (X축 좌우로 좁은 간격 바들바들 떨림. 0.1초 주기)
  - 텍스트: 타이핑 중간 0.2~0.3초 딜레이(말 더듬기). 말풍선 꼬리나 텍스트 폰트에 약한 Shake 효과 적용.
- **State 03. Annoyed (짜증/극대노):**
  - 모션: 위협적 확대 (캐릭터 스케일이 1.05배로 확 커지며 유저를 압박)
  - 텍스트: 평소보다 2배 빠른 0.02초 간격으로 출력. 화를 내는 키워드는 **Red & Bold** 처리되며 화면(Camera) 진동 동반.

**3. 핵심 기믹 타격감 연출 (Core Gimmick Impact)**

**3.1. PASS (통과) 도장 연출**

- **도장 낙하 (Spring):** 도장 오브젝트가 크기 6.0배 ➡️ 1.0배, 회전각 -45도 ➡️ -15도로 용수철처럼 강하게 내리꽂힘 (소요 시간: 0.4초, Ease-Out-Back).
- **타격 진동 (Shake):** 도장이 바닥에 닿는 순간, 서류 UI 및 화면 전체가 X축/Y축으로 격렬하게 진동 (0.4초).

**3.2. EJECT (귀가 조치) 셔터 연출**

- **경고 효과:** EJECT 레버 조작 즉시 화면 전체에 붉은색 경고등 비네팅이 점멸함 (4.0초 지속).
- **셔터 하강:** 화면 상단에서 철제 셔터가 0.15초 만에 위에서 아래로 매우 빠르게 닫힘 (Ease-In).
- **타격 진동:** 셔터가 바닥에 닫히는 즉시 쾅! 하는 효과음과 함께 화면 전체 진동 (0.5초).

**4.2. 메인 씬 (Main Scene) - 인게임 핵심 흐름**

본 항목은 메인 씬에 진입했을 때부터 근무가 종료될 때까지의 시간적 흐름(Sequence)과 상황별 연출 트리거를 정의합니다. (※ 각 요소의 상세 이펙트 및 수치는 2, 3항목의 글로벌 규칙을 따릅니다.)

**[ 발동 조건 (Trigger) ]**

- 인트로 씬 텍스트 종료 후, 또는 일일 정산(엔딩/다음 날) 화면에서 전환 시 Fade-in 되며 시작.

**[ 씬(Scene) 시퀀스 흐름도 ]**

**Phase 1. 씬 진입 및 환경 세팅 (Day Start)**

- **시각:** 화면 Fade-in (0.5초). 화면 중앙에 [ DAY - N : 근무 시작 ] 텍스트가 나타났다 사라짐.
- **청각:** 로우파이(Lo-Fi) 감성의 칙칙한 오피스 메인 BGM 재생 시작.

**Phase 2. 일일 업무 브리핑 (상급자 등장)**

- **스폰:** [특수] 상급자 NPC가 데스크 앞에 등장.
- **대화:** 타이핑 연출로 오늘의 특이사항(위장 용사 특징, KPI 목표 등) 브리핑 진행.
- **퇴장:** 대화가 끝나면 화면에 거대한 '대외비' 도장이 찍히는 연출(3.1 항목 응용)과 함께 상급자 퇴장.

**Phase 3. 메인 심문 루프 (Core Gameplay Loop)** *이 단계는 하루 할당량(또는 제한 시간)이 끝날 때까지 반복됩니다.*

- **① NPC 스폰:** 화면 밖(좌/우)에서 데스크 중앙으로 NPC가 걸어 들어옴 (Walk-in 애니메이션).
- **② 서류 제출:** NPC가 자신의 사원증/허가증 팝업을 화면에 띄움. (이때 NPC는 State 01: Normal 상태)
- **③ 상호작용 & 심문:** 유저가 매뉴얼을 열고 채팅으로 모순을 지적함. (대화 내용에 따라 NPC의 상태가 Flustered 또는 Annoyed로 실시간 변화 - 2.4 항목 참조)
- **④ 판정 (Action):** 유저의 최종 선택에 따라 핵심 기믹 연출 발생.
  - [PASS 도장 클릭 시] 👉 3.1 항목의 타격감 연출 발생. NPC가 인사 후 화면 밖으로 걸어서 퇴장.
  - [EJECT 레버 클릭 시] 👉 3.2 항목의 셔터 하강 및 경고등 연출 발생. NPC 비명과 함께 화면 밑으로 강제 퇴장.

**Phase 4. 근무 종료 (Day End)**

- **트리거:** 하루에 배정된 NPC 검문이 모두 끝나거나 근무 시간이 종료되었고, 상급자 NPC와의 모든 스크립트 대화를 마쳤을 때.
- **연출:** 메인 BGM이 서서히 꺼지고 (Fade-out), 아날로그 알람 시계 소리(따르릉!) SFX 발생.
- **전환:** 화면이 0.5초간 Fade-out 되며 암전, 이후 4.3. 엔딩(일일 정산) 씬으로 자동 로드됨.

**4.4. 일일 정산 및 상점 연출 (Monitor Phase)**

본 항목은 하루의 근무가 종료된 후, 물리적인 씬(Scene) 이동 없이 데스크 좌측 하단의 모니터 화면으로 카메라가 줌인(Zoom-in)하여 진행되는 정산 및 상점 이용 시퀀스를 정의합니다.

**[ 발동 조건 (Trigger) ]** 4.2 항목의 Phase 4 (근무 종료) 연출 직후 진행.

**[ 시퀀스 흐름도 (Sequence) ]**

**Phase 1. 모니터 줌인 및 화면 전환 (Zoom-in)** **시각 연출:** 카메라가 좌측 하단의 사원증 디스플레이(모니터) 영역으로 빠르게 줌인(Zoom-in)됨 (소요 시간: 0.4초, Ease-In-Out). 확대가 완료되면 화면 전체가 '마왕물산 사내 OS' 바탕화면 이미지로 덮임.

- **청각 연출:** 구형 모니터가 켜지는 소리 (삐빅- 웅~).

**Phase 2. 결산 내역 출력 (Settlement Terminal)**

- **시각 연출:** 모니터 화면 중앙에 터미널 창(또는 영수증 UI)이 팝업됨. \* **텍스트 연출:** 오늘 번 일당, 차감된 벌금, 뇌물 수익 등이 글로벌 규칙(0.05초 간격)에 따라 타자기가 쳐지듯 한 줄씩 출력됨. (글로벌 규칙에 따라 마우스 클릭시 텍스트 출력 연출 스킵 가능)
- **청각 연출:** 기계적인 키보드 타자음.

**Phase 3. 분기 선택 버튼 노출**

- 텍스트 출력이 모두 끝나면, 터미널 창 하단에 2개의 OS 스타일 버튼이 나타남.
- **버튼 A [ 사내 복지 상점(자판기) 접속 ]** / **버튼 B [ 시스템 종료 (다음 날 출근) ]**

**Phase 4. 버튼 조작에 따른 액션**

- **[ 사내 복지 상점 ] 클릭 시:**
  - 띠링! 하는 알림음과 함께, 모니터 화면 내에 **[ ☕ 믹스 커피 (심문 횟수 +2) : 50G ]** 아이템이 있는 상점 UI 창이 새롭게 플로팅 됨 (2.3 모달 공통 연출 적용 ).
  - 구매 완료 후 상점 창을 닫으면 다시 Phase 3의 대기 상태로 돌아감.
- **[ 시스템 종료 ] 클릭 시:** \* **시각 연출:** 브라운관 모니터가 꺼지는 이펙트(화면이 가로줄로 압축되며 까맣게 꺼짐) 발생 후, 0.5초간 화면 전체 암전.
  - **로직 처리:** Day 변수가 +1 증가하며 카메라가 다시 원래 데스크 뷰로 줌아웃(Zoom-out)됨. 즉시 4.2. 메인 씬 - Phase 1 (근무 시작)으로 루프(Loop) 됨.

  - 기준점(Transform Origin): 화면 좌측 하단 X: 14%, Y: 80.5% 지점 (사원증 모니터 위치)
