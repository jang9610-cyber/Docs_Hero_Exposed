## 📑 3. 필수 관리 플로우차트 목록
1.  **전체 게임 루프 (Main Loop)**: 타이틀 -> 메인 데스크 -> 심문 시작 -> 결과 정산 -> 다음 날짜(Day) 연동.
2.  **심문 상호작용 (Interrogation Logic)**: 질문 입력 -> AI 응답 생성 -> 수치 가중치 판단 -> 변화 트리거.
3.  **증거물 대조 시스템 (Evidence Check)**: 증거물 선택 -> AI 답변 텍스트 분석 -> 모순 판정(성공/실패).
4.  **시스템 설정 (Option Flow)**: 사운드, 그래픽, 언어 변경 시의 로직 전이 및 세이브 데이터 반영.
