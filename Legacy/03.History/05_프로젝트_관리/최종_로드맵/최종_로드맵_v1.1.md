# [용사님, 들켰죠?] Refined Project Roadmap (v1.1)

## 📌 주요 개선 사항
1.  **역할군 명확화**: PM(기획/프롬프트/밸런스), Dev(시스템/API/통합), Design(UI/캐릭터/FX)으로 상세 분리.
2.  **프롬프트 튜닝 세션 추가**: Gemini API의 핵심은 '프롬프트'입니다. 단순 개발 외에 PM과 Dev가 협업하여 NPC의 '지능'을 조정하는 시간을 배치했습니다.
3.  **병목 구간 조정**: 3주차 프로토타입 마감 전후의 QA 및 폴리싱 기간을 보강했습니다.
