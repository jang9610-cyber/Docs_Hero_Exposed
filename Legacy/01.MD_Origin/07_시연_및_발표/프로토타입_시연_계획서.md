## 🎬 1. 전체적인 씬 루프 (Scene Flow Overview)

> 게임의 전체적인 흐름을 빠르게 보여주어 발표 대상자가 게임 구조를 직관적으로 파악할 수 있도록 합니다.

**시연 시퀀스**: `메인화면` → `오프닝 컷씬` → `게임 플레이` → `엔딩 컷씬` → `게임 종료 화면`

### 시연 목적
- 게임 내 **씬 전환의 전체 흐름**을 일련의 순서로 시연
- 각 씬이 자연스럽게 연결되는 **루프 구조**를 빠르게 체험

### 씬별 역할 요약

| 씬 | 내용 |
| :--- | :--- |
| **메인화면** | 게임 진입 전 타이틀 화면 |
| **오프닝 컷씬** | 게임의 서사적 배경 및 세계관 소개 (총 8장 구성) |
| **게임 플레이** | 핵심 게임플레이 루프 진행 (심문 → 정산 → 상점) |
| **엔딩 컷씬** | 결과에 따른 엔딩 연출 (총 3장 구성) |
| **게임 종료 화면** | 게임 종료 및 결과 확인 |

## 🖥️ 2. 게임 화면 UI 기능 상세 설명 (UI Features)

> 게임 플레이 씬에 구현된 UI 팝업 기능을 세부적으로 설명합니다.

### 2-1. UI 팝업 목록

#### 📋 ID Card 디스플레이
- **기능**: 현재 면담 중인 NPC의 기본 정보(이름, 종족, 소속 등)를 출력
- **목적**: 플레이어가 면담 전 대상에 대한 기초 정보를 파악하고 심문에 활용
- **현황**: ✅ 구현 완료

#### 📝 메모장 보드
- **기능**: 매 일차마다 새로운 특이사항(Notice) 정보가 표시됨
- **핵심 포인트**: 해당 정보는 **가짜 마족에게는 전달되지 않으므로** 진위 판별의 핵심 단서로 활용 가능
- **현황**: ✅ 구현 완료 (시스템 프롬프트와 실시간 연동)

#### 📖 종족별 매뉴얼
- **기능**: 게임 세계관에 등장하는 다양한 종족의 기본적인 특징 및 설명을 기술한 참고 문서
- **추리 단서로서의 기능**: 정보를 비교하여 위화감을 찾는 추리 단서로 활용 가능
- **현황**: ⚠️ 매뉴얼 열람 기능은 구현됨 / **추리 단서 연동 기능은 미구현** (향후 개발 예정)

## ⚙️ 3. 게임 시스템 상세 설명 (Game Systems)

> 게임의 핵심 메커니즘을 구체적으로 설명합니다.

### 3-1. LLM 기반 NPC 대화 시스템

- **기술**: LLM(Large Language Model) API를 활용한 실시간 AI NPC 대화
- **초기 동작**: NPC 등장 시 **상황 또는 기본 정보와 관련된 주제**로 자연스럽게 대화를 시작
- **플레이어 역할**: AI NPC와 **자유롭게 대화**하며 해당 NPC가 **진짜 마족인지 가짜 마족인지** 판별
- **현황**: ✅ 구현 완료

### 3-2. 진짜와 거짓의 판별 (Pass / Eject)

- **판별 방법**:
  - NPC와의 대화 흐름에서 **모순점을 발견**하거나, 의도적으로 **유도 발언**을 통해 진위를 확인
  - 메모장 정보, ID Card 정보 등과 NPC 답변을 **교차 대조**
- **최종 판정**: 정체를 파악했다고 판단될 경우 아래 버튼으로 결정
  - ✅ **Pass 버튼**: 통과 (진짜 마족으로 판단)
  - ❌ **Eject 버튼**: 추방 (가짜 마족으로 판단)
- **현황**: ✅ 구현 완료

### 3-3. 아이템 사용 시스템

- **구매 시점**: **일차 종료 후** 접근 가능한 **상점(Shop)** 에서 구매
- **재화**: 심문 결과 정산에서 획득한 인사고과 재화로 구매
- **현재 구현된 아이템**: 시연을 위한 기본 아이템 구현 완료
- **현황**: ⚠️ 시연용 아이템 구현 완료 / **개발 과정에서 내용 수정 및 삭제 가능**

### 3-4. NPC 감정 변화 및 행동 유도 시스템

- **개요**: 플레이어의 행동에 따라 NPC의 **내부 감정 수치**가 변화하며 이에 따른 감정 판정이 이루어짐
- **수치 변화 트리거**:
  - 플레이어의 **특정 질문 또는 유도 발언**
  - 인게임 **아이템 사용** (예: 에너지 드링크 등)
- **감정 표현**: 스트레스 수치 변화에 연동된 **NPC 표정 변화 및 화면 연출** 반영
- **현황**: ✅ 구현 완료

## 📌 4. 시연 시 유의사항 (Notes for Demo)

- 종족별 매뉴얼의 **추리 단서 연동은 미구현 상태**임을 사전 안내할 것
- 상점 아이템은 **시연용으로 구성**된 버전이며 최종 빌드에서 변경될 수 있음
- AI NPC 응답은 **JSON 형식으로 파싱**되어 처리되므로 네트워크 상태에 유의
