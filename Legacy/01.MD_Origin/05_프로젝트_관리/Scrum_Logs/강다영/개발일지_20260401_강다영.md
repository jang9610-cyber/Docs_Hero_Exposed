### 🛠️ Trouble Shooting
- **AI 응답 JSON 형식 불일치 문제**:
    - **문제**: AI에게 JSON 형식으로 답변하라고 요청했으나, 종종 텍스트가 섞여 나오거나 형식을 무시하여 시스템 스트레스 수치 연동이 중단되는 현상.
    - **해결**: 기술 매니저의 자문을 받아 `CreateChatCompletionRequest` 내 `ResponseFormat` 프로퍼티를 `ResponseType.JsonObject`로 설정하여 해결 방방을 마련함. 이를 통해 API 레벨에서 JSON 출력을 보장받게 됨.

> **[운영 과제]** 주요 기능과 흐름은 완성되었으나, AI와의 대화 품질이 곧 게임의 개성임. 매니저님께서 제공한 프롬프트 고도화 전략을 충실히 이행하여 완성도 높은 프로토타입을 마감할 것.
