## 💡 오후 기술 자문 및 주요 안건 (Technical Consultation)

### **AI 프롬프트 고도화 및 JSON 응답 제어**
- **현황 및 문제점**:
    - AI가 대화 내용을 기반으로 스트레스 수치를 가변적으로 조정하도록 JSON 형식을 요청했으나, AI가 의도된 형식을 지키지 않거나 작동이 불안정한 문제 발생.
- **기술 매니저 자문 (오후 4시)**:
    - **키워드 기반 로직**: 대화 로그에서 특정 키워드를 추출하여 스탯을 증감시키는 보조 로직 검토.
    - **프롬프트 강화**: AI가 대화 내용을 지속적으로 기억하고, 설정된 형식을 엄격히 준수하도록 시스템 프롬프트 구조 고도화.
    - **응답 형식 강제**: API 호출 시 `ResponseFormat` 설정을 통해 JSON 출력을 명시적으로 강제하는 방안 제언.
- **의사결정 및 적용**:
    - `CreateChatCompletionRequest` 설정에 `ResponseFormat { Type = ResponseType.JsonObject }`를 적용하여 JSON 응답의 안정성을 확보하기로 결정.
    - **예문 코드**:
      ```csharp
      var request = new CreateChatCompletionRequest
      {
          Model = "helpy-pro",
          Messages = new List<ChatMessage>
          {
              new ChatMessage 
              { 
                  Role = "user", 
                  Content = "사용자 정보를 JSON으로 만들어줘" 
              }
          },
          ResponseFormat = new ResponseFormat 
          { 
              Type = ResponseType.JsonObject  // JSON 형식 강제
          }
      };
      ```
    - 해당 기술 적용 후, 수요일(금일)부터 본격적인 프롬프트 고도화 사전 준비 실시.
