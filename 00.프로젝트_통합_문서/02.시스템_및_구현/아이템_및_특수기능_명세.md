# 아이템 및 특수 기능 명세 (Item & Special Function Specification)

> **[문서 목적]** 본 명세서는 게임 내 상점(자판기) 아이템의 고유 효과 메커니즘과, 신규 유저를 위한 튜토리얼 컨텐츠가 화면에 어떻게 표시되고 제어되는지를 정의하는 기획 명세서입니다.

## 0. 기획 의도 (Design Intent)
> **[아이템]**은 단순히 게임을 쉽게 만드는 것이 아니라, LLM 기반 NPC의 강력한 '페르소나 락'을 뚫거나 높은 난이도의 대화에서 전략적 돌파구(스트레스 조작, 진실 강제)를 마련해 주는 조커 카드입니다.
> **[튜토리얼]**은 복잡한 추리 루프(정보 읽기 ➔ 비교 ➔ 확신)를 유저가 직관적으로 학습할 수 있도록 화면을 분할해 인게임 설정(사내 지침)에 자연스럽게 녹여내는 것을 목적으로 기획되었습니다.

---

## 1. 상점 아이템 효과 명세 (Shop Items)

사내 복지 상점(자판기)에서 구매 가능한 3종의 아이템은 심문(대화) 세션 중 `ItemManager`와 `NPCChatManager`를 통해 인게임 로직에 직접적인 영향을 미칩니다.

| 아이템명 | 가격 | 핵심 효과 (Effect) | 상세 메커니즘 및 툴팁 (Tooltip) |
| :--- | :--- | :--- | :--- |
| **고농축 에너지 드링크** | 150G | **턴 연장 (+2)** | 사용 시 현재 진행 중인 대상 NPC와의 잔여 심문 가능 턴(대화 횟수)을 즉시 2회 증가시킵니다. |
| **최고급 연초** | 200G | **스트레스 강제 주입 (+50)** | 사용 시 대상 NPC에게 담배 연기를 뿜어 스트레스 수치를 즉시 50 증가시킵니다. NPC가 더 빨리 본성을 드러내게 유도합니다. |
| **불법 최면 어플** | 450G | **진실 강제 (프롬프트 제어)** | 사용 직후 플레이어가 입력하는 단 1회의 질문에 한해, NPC가 종족이나 설정에 관계없이 무조건 진실만을 대답하도록 시스템 프롬프트(LLM)에 강력한 제어 지시를 강제합니다. |

> **[아이템 사용 로직]**
> * 아이템은 메인 씬의 좌측 하단 인벤토리 슬롯(`Grp_Inventory`)에 최대 3개까지 보관됩니다.
> * 아이템 클릭 시 즉각 소모되며, 효과 연출(VFX/사운드)과 함께 기능이 발동됩니다.

## 2. 튜토리얼 시스템 명세 (Tutorial System)

첫 플레이 시 또는 플레이어가 우측 상단 `[?]` 아이콘(`Btn_Tutorial`)을 클릭했을 때 호출되는 게임 규칙 가이드 시스템입니다.

### 2.1 튜토리얼 상호작용 로직
- **호출 방식**: 메인 화면에서 `Btn_Tutorial` 클릭 시 `Canvas_Tutorial` 패널을 활성화(`SetActive(true)`)
- **닫기 방식**: 화면 전역을 덮고 있는 **투명 배경 버튼(`Button` 1920x1080, Alpha 0) 클릭 시 즉시 닫힘(`SetActive(false)`)**. 명시적인 [X] 닫기 버튼은 존재하지 않습니다.

### 2.2 튜토리얼 표시 정보 (UI 패널별 내용)
튜토리얼은 5개의 반투명 패널 구역으로 나뉘어 각 시스템의 활용법을 안내합니다.

| 구역명 (UI Panel) | 가이드 내용 (Tutorial Content) | 비고 |
| :--- | :--- | :--- |
| **메모판 (MenoPanel)** | **상관 지침서 확인** — 일일 정산 시 또는 데스크에서 확인 가능한 일일 메모(지침)입니다. 오늘의 특이사항이나 임무 목표가 적혀 있으므로 꼼꼼히 확인해야 합니다. | 분홍 반투명 패널 |
| **신분증 (DiaplayPanel)** | **서류 대조 작업** — NPC가 제시한 출입 허가증(사원증)의 기재 내용(소속, 종족, 이름)과 실제 매뉴얼 정보를 대조하여 위조 여부를 판단합니다. | 초록 반투명 패널 |
| **NPC 렌더링 (NpcPanel)** | **외형 및 태도 관찰** — 상대의 옷차림, 땀, 표정 등 겉모습 특징을 살펴보고 대화를 통해 모순점이 없는지 심문하십시오. | 회색 반투명 패널 |
| **입력창 (TextPanel)** | **직접 심문 (타이핑)** — 키보드를 통해 NPC에게 직접 질문을 던지거나 압박할 수 있습니다. 날카로운 질문은 상대의 스트레스를 높입니다. | 카키 반투명 패널 |
| **판정 도장 (ButtonPanel)** | **PASS / EJECT 판정** — 모든 정보를 종합하여 상대가 '진짜' 직원이라면 PASS 도장을, 마족이나 스파이라면 EJECT 레버를 당겨 추방하십시오. | 빨강 반투명 패널 |
| **아이템 (ItemPanel)** | **인벤토리 및 아이템** — 상점에서 구매한 아이템은 여기에 보관됩니다. 심문 중 필요할 때 클릭하여 즉시 사용할 수 있습니다. | 연흰색 반투명 패널 |
