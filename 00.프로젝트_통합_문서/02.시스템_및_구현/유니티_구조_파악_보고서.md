## 1. 씬(Scene) 구성

| 씬명 | 경로 | 역할 |
| :--- | :--- | :--- |
| **MainScene** | `Assets/01_Scenes/MainScene.unity` | 타이틀 & 메인 메뉴 |
| **CutScene** | `Assets/01_Scenes/CutScene.unity` | 인트로 컷신 |
| **GameScene** | `Assets/01_Scenes/GameScene.unity` | 핵심 심문 플레이 씬 |
| **SettlementScene** | `Assets/01_Scenes/SettlementScene.unity` | 일일 정산 + 상점 씬 |
| **EndScene** | `Assets/01_Scenes/EndScene.unity` | 최종 엔딩 분기 씬 |

## 2. 스크립트 전체 목록 (02_Scripts, 총 70개)

### 2.1. Manager (핵심 매니저)
| 스크립트 | 역할 |
| :--- | :--- |
| `GameManager.cs` | 게임 전체 상태(Day 진행, 판정, 엔딩 분기) 관리 |
| `DataManager.cs` | 세이브/세션/정적 데이터의 로드/저장 |
| `AudioManager.cs` | BGM, SFX 재생 관리 |
| `UIManager.cs` | 전역 팝업 레지스트리 및 표시 제어 |
| `ItemManager.cs` | 아이템 사용 효과 처리 |
| `SceneController.cs` | 씬 전환 로직 |
| `LoadingManager.cs` | 씬 로딩 UI 처리 |
| `CutSceneManager.cs` | 컷신 재생 제어 |
| `NPCChatManager.cs` | 프롬프트 조립 및 OpenAI API 통신 |
| `SettlementController.cs` | 정산 씬 전용 로직 |

### 2.2. Data (데이터 구조체 & ScriptableObject)
| 스크립트 | 역할 |
| :--- | :--- |
| `SaveData.cs` | 영구 세이브 데이터 구조체 (골드, 엔딩 분기 조건 등) |
| `SessionData.cs` | 한 세션(하루) 동안의 임시 데이터 |
| `StaticData.cs` | 기획 수치 정적 데이터 (아이템 가격, 스트레스 기준 등) |
| `CharacterData.cs` | NPC 캐릭터 정보 (종족, 타입, 이름 등) |
| `UIPopupData.cs` | 팝업 활성화 상태 데이터 |
| `CutsceneStep.cs` | 컷신 단계별 데이터 |
| `DataSyncAttribute.cs` | SaveData ↔ SessionData 동기화 어트리뷰트 |

### 2.3. Data/Prompt (ScriptableObject 프롬프트)
| 스크립트 | 역할 |

| `RacePrompt.cs` | 종족별 페르소나 지침 |
| `DailyMemoPrompt.cs` | 일일 특이사항 지침 |
| `PromptText.cs` | 프롬프트 텍스트 데이터 컨테이너 |
