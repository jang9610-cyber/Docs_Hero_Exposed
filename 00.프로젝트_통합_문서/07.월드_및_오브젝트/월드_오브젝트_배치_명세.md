# 월드 오브젝트 배치 명세 (World Objects & Background Specification)

> **[문서 목적]** 본 명세서는 GameScene 내 배경 스프라이트, NPC 프록시, 배경 NPC 스포너 등 월드 오브젝트의 구성과 역할을 정의하는 기술 명세서입니다.

## 0. 기획 의도 (Design Intent)
> 월드 오브젝트 레이어는 심문이 이루어지는 '경비실 데스크' 공간을 2D로 표현하는 모든 배경 요소를 포함합니다.
> 핵심 게임플레이(심문/UI)가 전면부(Front)에서 이루어지는 반면, 월드 오브젝트들은 후면부(Back)에서 공간감과 생동감을 담당합니다. **[배경 NPC 스포너(`BG_NPC_Spawner`)]**는 실제 심문 대상과 무관한 배경 캐릭터들을 주기적으로 스폰하여 '회사 로비처럼 사람들이 오가는 공간'이라는 생동감 있는 분위기를 연출합니다.

---

## 1. 오브젝트 목록 (`====WorldObjects====`)

GameScene 내 `====WorldObjects====` 그룹 하위에 배치된 모든 월드 오브젝트 목록:

| 오브젝트명 | 스크립트 | Layer | 역할 |
| :--- | :--- | :--- | :--- |
| `WO_LowBG` | ─ | Default | 배경 스프라이트 렌더러. 로비/경비실 배경 이미지 표시. |
| `WO_NPC` | `NPCWorldProxy.cs` | Default | 심문 중인 현재 NPC의 스프라이트를 씬에 표시하는 프록시. |
| `BG_NPC_Spawner` | `BackgroundNPCSpawner.cs` | Default | 배경에서 오가는 엑스트라 NPC들을 주기적으로 스폰/디스폰. |

---

## 2. 각 오브젝트 상세 명세

### 2.1. WO_LowBG (배경 이미지)
- **컴포넌트**: `Transform`, `SpriteRenderer`
- **역할**: 게임 배경(경비실 데스크 후면)의 메인 배경 스프라이트를 렌더링합니다.
- **라이팅**: `====Lighting2D====/Light_LowBG` (`Light2D`)의 직접 조명을 받습니다.

### 2.2. WO_NPC (NPC 월드 프록시)
- **스크립트**: `NPCWorldProxy.cs`
- **컴포넌트**: `Transform`, `SpriteRenderer`, `NPCWorldProxy`
- **역할**: 심문 대상 NPC의 스프라이트를 씬의 3D 공간에 표시하는 '프록시(대리인)' 역할을 합니다. 실제 NPC 데이터(`CharacterData`)는 `DataManager`가 관리하며, `NPCWorldProxy`는 단순히 스프라이트 이미지와 등장/퇴장 애니메이션만을 담당합니다.
- **NPC 등장/퇴장 연출**: `VFXManager_NPC.cs`로부터 이벤트를 수신하여 슬라이드인/슬라이드아웃 트윈 애니메이션을 재생합니다.
- **라이팅**: `====Lighting2D====/Light_NPC` (`Light2D` + `LightSwing`)의 조명을 받아, 진자 운동하는 조명 효과가 NPC에게 자동 적용됩니다.

### 2.3. BG_NPC_Spawner (배경 NPC 스포너)
- **스크립트**: `BackgroundNPCSpawner.cs`
- **컴포넌트**: `Transform`, `BackgroundNPCSpawner`
- **역할**: 심문과 무관한 배경용 엑스트라 NPC들을 일정 주기와 랜덤 포지션에 스폰하고 특정 시간이 지나면 디스폰합니다. 마왕물산 로비에 다양한 직원들이 오가는 것처럼 시각적 생동감을 부여합니다.
- **주의**: 배경 NPC는 실제 심문 대상(`WO_NPC`)과 완전히 독립된 별도 오브젝트입니다.
